## 关于

爬虫练手项目，用于爬取本地部署的靶场网站，请勿用于真实环境！

## 使用方法

### 1. 安装依赖

确保已安装Python 3.x，并执行以下命令安装依赖：

> 推荐使用虚拟环境
> 
> ```shell
> python -m venv .venv
> source ./.venv/Script/activate
> ```

安装依赖

```shell
pip install requests sqlite-worker
```

### 2. 配置参数

编辑 `config.py` 文件
